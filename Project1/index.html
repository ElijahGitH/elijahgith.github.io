Project 1: Hydration Tracker (Humans Interactions Course)

Project Link

Open Hydration Tracker

Description

This hydration tracker was a project for my Humans Interactions course. The goal was to build a simple interface that makes it easy to log water intake and see progress over time.

Skills Learned

  • HTML structure and semantic layout
  • JavaScript fundamentals (functions, events, variables)
  • Linking pages and files together in a multi-file project
  • Building a basic GUI and improving usability
  • Connecting UI actions to logic in an app.js-style script
